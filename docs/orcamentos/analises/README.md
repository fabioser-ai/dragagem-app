# Registro Oficial das Análises de Orçamento

## Finalidade

Indexar as análises individuais de arquivos Excel e impedir colisões de nomes ou registros genéricos.

## Convenção

Cada análise deve ser identificada pelo nome completo do arquivo Excel, incluindo extensão. O documento Markdown correspondente deve preservar esse nome de forma reconhecível e inequívoca.

Exemplo de fonte: `D_004_2026 - SABESP.xlsx`.

## Registro existente

A primeira engenharia reversa vertical permanece preservada em:

- `docs/knowledge/orcamentos/001_DESAGUAMENTO_BAGS_SABESP.md` — fonte analisada: `D_004_2026 - SABESP.xlsx`.

Esse documento histórico deverá ser reconciliado com o padrão atual sem perda de conteúdo ou rastreabilidade.

## Regra de inclusão

Uma análise somente deve ser listada aqui após:

1. leitura integral do Excel;
2. documentação conforme a metodologia oficial;
3. confirmação de escrita e releitura;
4. declaração explícita de status e dúvidas pendentes.

A inclusão neste índice não equivale à consolidação de suas descobertas.
